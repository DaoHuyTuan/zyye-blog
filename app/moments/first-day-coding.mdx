# Ngày đầu tiên học lập trình

Còn nhớ ngày đầu tiên bước vào lớp học **Programming Fundamentals** tại FPT University. Cảm giác hồi hộp, lo lắng nhưng cũng rất háo hức.

## Dòng code đầu tiên

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```

Khi thấy dòng chữ "Hello World!" hiện lên trên màn hình đen, tôi biết mình đã tìm thấy **passion** của đời mình.

## Cảm nhận

- **Thú vị**: Việc tạo ra thứ gì đó từ con số 0
- **Thách thức**: Debug những lỗi syntax đầu tiên
- **Hy vọng**: Một ngày nào đó sẽ trở thành developer giỏi

> "Every expert was once a beginner. Every pro was once an amateur."

Từ ngày đó, coding không chỉ là môn học mà đã trở thành một phần không thể thiếu trong cuộc sống của tôi.
